À propos de moi...

Conducteur offset depuis plus de 20 ans, je travaillais à l'imprimerie Bertola qui est en voie de restructuration et licencié économique depuis le 13 juillet 2020.
Comme vous le verrez dans mon CV, j'ai été formé et ai travaillé dans l'Imprimerie Laffont puis dans l'Imprimerie Bertola.
Durant cette période, j'ai obtenu deux BEP, le premier dans la PAO, le second en impression avec un Compte Fil d'Or.
Souhaitant développer mes compétences professionnelles dans le domaine des nouvelles technologies, j'ai acquis mon Baccalauréat Professionnel : Réalisation de Produits Imprimés et Plurimédia Option B Productions Imprimées.
Je l'ai décroché par l'intermédiaire d'une Validation de mes Acquis de l'Expérience, afin d'acquérir le niveau 4.
Par la suite, j'ai utilisé mon compte CPF, pour étudier l'anglais durant une année chez Wallstreet English.
Ayant fait le tour dans mon précédent secteur professionnel, je suis très désireux d'entamer, aujourd'hui, une reconversion dans le métier de Développeur Web.
Pour se faire, j'ai profité du confinement pour commencer à apprendre le Web, le code HTML 5 et CSS 3 via Openclassrooms.
Je suis très motivé, tout disposé à continuer à apprendre et découvrir ce nouveau métier qui évolue sans cesse.
Je souhaite suivre cette formation, acquérir des compétences nouvelles, pour changer de voie professionnelle.
